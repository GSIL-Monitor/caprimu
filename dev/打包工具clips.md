##Babel安装 
命令行下，Babel的安装命令如下。

$ npm install --global babel-cli
$ npm install --save babel-preset-es2015
然后在当前目录下，新建一个配置文件.babelrc。

// .babelrc
{
  "presets": ['es2015']
}
